## 평가 항목 및 기준 (README §4-1 기준)

1. 제구력 (command) — 눈매의 선명도(4점), 눈의 안정감(3점), 눈-눈썹 간격(3점)
   - 눈동자가 또렷하고 초점이 분명할수록 높은 점수
   - 흐릿하거나 불안정한 눈빛은 감점

2. 구위 (stuff) — 턱·하관 라인(4점), 광대뼈 돌출도(3점), 코의 크기·높이(3점)
   - 각지고 단단한 하관, 적당한 광대, 높고 넓은 콧볼 = 파워
   - 둥글거나 평면적인 하관은 감점

3. 멘탈 (composure) — 이마 넓이·높이(4점), 인중 깊이·길이(3점), 미간 간격(3점)
   - 넓고 높은 이마, 뚜렷한 인중, 적절한 미간 = 정신력
   - 좁은 이마·짧은 인중·과도하게 좁거나 넓은 미간은 감점

4. 지배력 (dominance) — 눈썹 기세·농도(4점), 얼굴 좌우 균형(3점), 이목구비 비율(3점)
   - 짙고 힘 있는 눈썹, 균형 잡힌 이목구비 = 카리스마
   - 옅은 눈썹·심한 비대칭은 감점

5. 운명력 (destiny) — 전체 인상 조화도(4점), 얼굴형(3점), 귀·입 조화(3점)
   - 첫인상에서 풍기는 "되는 사람" 느낌, 귀 크기, 입 모양
   - 조화롭지 않거나 기운이 흩어진 인상은 감점

## 점수 배분 원칙
- 각 항목 score: 0~10 정수 (5 = 평균, 8 이상 = 상위 10%)
- 한 투수의 5개 항목 평균이 5±1.5 범위가 되도록 조정하세요
- 모든 항목을 7~8점으로 주는 것은 엄격히 금지합니다

Return ONLY valid JSON, no markdown fences.
